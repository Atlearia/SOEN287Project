# Smart Course Companion – Deliverable 1 (Frontend)

## Course
SOEN 287 – Web Programming

## Deliverable Scope
This deliverable focuses **ONLY on the frontend** using:
- HTML5
- CSS3

---

## Implemented User
- Student
- Instructor / Admin

---

## Pages

### Authentication Pages
- Login page
- Register page
- Role selection (Student / Admin)

### Student Pages
- Student dashboard (all courses overview)
- Course details page
- Assessment list per course
- Upcoming assessments page

### Instructor/Admin Pages
- Admin dashboard
- Course creation page
- Assessment category & weighting page
- Usage statistics
- Enable or disable courses. 

---

## Features Implemented (UI Only)
- Add/edit/delete courses
- Add/edit/delete assessments
- Mark assessments as completed/pending
- Display current course average
- Progress bars for course completion
- Charts or visual summaries
- Responsive navigation between pages

- Navigation bar present on all pages
